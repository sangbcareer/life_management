# AWS클라우드([[Cloud]]) 개요 - 리전&AZ


## 1. AWS의 역사

2002년 : Amazon 내부에서 시작, IT부서를 외주로 운영할 수 있다는 것을 알았다.
2003년 : Amazon의 핵심역량이 되어, 다른 사업을 위한 IT 비즈니스 가능성 확인
2004년 : SQS 첫 런칭
2006년 : SQS, S3&EC2를 포함한 서비스 재런칭
2007년 : 유럽에 런칭

현시점에 있어서, 클라우드 공급자 순위에서 가장 큰 매출규모를 차지하고 있음(9년 연속)


## 2. AWS 클라우드 사용 사례

IT 서비스를 만드는 데에 있어 거의 모든 것을 할 수 있음

- 복잡하고 확장 가능([[expandable]])한 애플리케이션([[Application]]) 구축
- 다양한 산업 분야에 적용 (맥도널드, 넷플릭스, 산토리)
- 기업 IT 이전, 백업&스토리지, 빅 데이터 분석
- 웹사이트 호스팅([[Hosting]]), 모바일&소셜 애플리케이션
- 게이밍
